<p align="center"><sub><em>
The interesting thing about being early to a paradigm shift is you get to name the primitives.<br />
I build the missing substrate for autonomous agents — the things every serious agent stack will eventually need.<br />
Not wrappers. Not yet another framework. Foundations.<br />
Diff engines that find the exact decision fork between two runs.<br />
Staleness monitors that alert when facts agents relied on have changed.<br />
Attestation layers that prove what an agent actually did.<br />
Belief-repair systems that catch when game worlds contradict themselves.<br />
Twelve focused Python libraries. One thesis: the agentic era needs better substrate.
</em></sub></p>

## Now

Building **12 open-source Python libraries** for the full agentic AI infrastructure stack. Each ships with a CLI, REST API, and MCP server interface. All MIT-licensed.
